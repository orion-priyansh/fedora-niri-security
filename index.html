01

Pre-Installation Preparation

⚠
Back up Windows first. Use Windows Backup or clone your drive. Dual boot partitioning carries risk of data loss if done incorrectly.
Windows 11 — Disable Fast Startup & Secure Boot

Fast Startup causes filesystem lock issues on shared partitions. Secure Boot may block Fedora's bootloader (though Fedora supports it — disable if problems arise).

# In Windows PowerShell (Admin):
powercfg /h off            # Disable hibernation / fast startup

# Then: Settings → System → Recovery → Advanced Startup
# → UEFI Firmware Settings → Disable Secure Boot (if needed)
Shrink Windows Partition

Allocate at least 80–120 GB for Fedora (more if storing VMs, datasets, or blockchain nodes).

# Windows → Disk Management → Right-click C: → Shrink Volume
# Recommended layout:
  EFI    →  512 MB  (already exists, shared)
  Windows →  keep existing
  [FREE] →  100–200 GB  (for Fedora)
Create Fedora Bootable USB
# Download Fedora Workstation 40+ (or KDE spin) ISO
# https://fedoraproject.org/workstation/download/

# Flash with Fedora Media Writer (Windows) or on Linux:
sudo dd if=Fedora-Workstation-Live.iso of=/dev/sdX bs=4M status=progress
ℹ
Boot the USB in UEFI mode (not legacy/BIOS). Check your firmware boot menu — usually F12, F10, or Del at startup.
02

Fedora Installation & Dual Boot

Anaconda Installer — Partition Setup

Use Custom Partitioning (Blivet-GUI or manual). Do NOT let the installer auto-configure if Windows is present.

  /boot/efi  →  512 MB   EFI  (select existing — DO NOT reformat)
  /boot      →  1 GB     ext4
  /          →  60 GB    btrfs  (Fedora default, supports snapshots)
  /home      →  rest     btrfs
  swap       →  8–16 GB  (or use zram — modern default)
⚡
Select the existing EFI partition and set its mount point to /boot/efi — do not format it. This preserves the Windows bootloader.
Post-Install — Verify GRUB Detects Windows
sudo grub2-mkconfig -o /boot/grub2/grub.cfg
sudo dnf install os-prober
# Enable os-prober in /etc/default/grub:
echo 'GRUB_DISABLE_OS_PROBER=false' | sudo tee -a /etc/default/grub
sudo grub2-mkconfig -o /boot/grub2/grub.cfg
Initial System Update
sudo dnf upgrade --refresh -y
sudo dnf install fedora-workstation-repositories -y
sudo dnf install @development-tools kernel-devel -y

Niri Wayland Compositor Setup

Niri is a scrollable-tiling Wayland compositor — excellent for multitasking across many terminal windows, recon outputs, and browser tabs simultaneously.

Install Niri & Dependencies
# Enable COPR repo for niri
sudo dnf copr enable yalter/niri -y
sudo dnf install niri -y

# Essential Wayland utilities
sudo dnf install waybar fuzzel foot mako wl-clipboard \
  xdg-desktop-portal-gnome pipewire wireplumber \
  polkit-gnome network-manager-applet -y

# XWayland for legacy X11 apps (Burp, older tools)
sudo dnf install xorg-x11-server-Xwayland -y
Niri Config — Security-Focused Layout
mkdir -p ~/.config/niri
nano ~/.config/niri/config.kdl
// ~/.config/niri/config.kdl — Security Workstation Layout input { keyboard { xkb { layout "us" } repeat-delay 250 repeat-rate 30 } touchpad { tap } } layout { gaps 8 center-focused-column "never" default-column-width { proportion 0.5; } focus-ring { width 2 active-color "#00e5ff" inactive-color "#1e2a36" } } binds { // Launcher, terminal, browser Mod+Return { spawn "foot"; } Mod+D { spawn "fuzzel"; } Mod+B { spawn "firefox"; } // Layout navigation Mod+H { focus-column-left; } Mod+L { focus-column-right; } Mod+J { focus-window-down; } Mod+K { focus-window-up; } Mod+Q { close-window; } Mod+F { maximize-column; } Mod+Shift+E { quit; } }
Waybar Status Bar for Security Recon
mkdir -p ~/.config/waybar
# In ~/.config/waybar/config.json — add modules:
"modules-right": ["network", "cpu", "memory", "temperature",
                   "battery", "clock", "tray"]
# Shows live CPU/RAM — essential when running scans/models
Set Niri as Login Session
# Log out → select "Niri" from SDDM/GDM session dropdown
# Or start manually from TTY:
niri-session

Core System Configuration

Enable RPM Fusion & Flathub
sudo dnf install https://mirrors.rpmfusion.org/free/fedora/rpmfusion-free-release-$(rpm -E %fedora).noarch.rpm
sudo dnf install https://mirrors.rpmfusion.org/nonfree/fedora/rpmfusion-nonfree-release-$(rpm -E %fedora).noarch.rpm
flatpak remote-add --if-not-exists flathub https://flathub.org/repo/flathub.flatpakrepo
Kernel Hardening & sysctl
sudo nano /etc/sysctl.d/99-security.conf
# Add:
net.ipv4.tcp_syncookies = 1
net.ipv4.conf.all.rp_filter = 1
net.ipv4.conf.all.accept_redirects = 0
net.ipv6.conf.all.accept_redirects = 0
kernel.dmesg_restrict = 1
kernel.kptr_restrict = 2
net.ipv4.ip_forward = 0
sudo sysctl --system
Python, Go, Node, Rust Toolchains
sudo dnf install python3 python3-pip python3-venv golang nodejs npm -y
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
# Go path:
echo 'export PATH=$PATH:$(go env GOPATH)/bin' >> ~/.bashrc
Docker & Podman for Isolated Environments
sudo dnf install podman podman-compose -y
# Docker (alternative):
sudo dnf config-manager --add-repo https://download.docker.com/linux/fedora/docker-ce.repo
sudo dnf install docker-ce docker-ce-cli containerd.io -y
sudo systemctl enable --now docker
sudo usermod -aG docker $USER

Privacy, OPSEC & Network Security

VPN & Traffic Isolation
# WireGuard (modern, fast VPN)
sudo dnf install wireguard-tools -y
# Or Mullvad / ProtonVPN with native clients:
# https://mullvad.net/download/vpn/linux

# OpenVPN (for older corporate/HTB/THM labs)
sudo dnf install openvpn network-manager-openvpn-gnome -y
Firewall with nftables
# Fedora uses firewalld (nftables backend)
sudo systemctl enable --now firewalld
sudo firewall-cmd --set-default-zone=drop
sudo firewall-cmd --permanent --add-service=ssh
sudo firewall-cmd --reload
# GUI management:
sudo dnf install firewall-config -y
Tor, Browser Isolation & Password Management
sudo dnf install tor torbrowser-launcher -y
sudo systemctl enable --now tor

# Bitwarden — FOSS password manager
flatpak install flathub com.bitwarden.desktop -y

# Firejail for sandboxing tools
sudo dnf install firejail -y
firejail --net=none burpsuite
SELinux (Fedora Default — Keep Enabled)
# Fedora ships with SELinux enforcing — do NOT disable it.
# Check status:
getenforce
Enforcing
# Audit denials (useful for debugging tool issues):
sudo ausearch -m avc -ts recent | audit2why

Niri Keybinds Reference

Mod + Return
Open terminal (foot)
Mod + D
App launcher (fuzzel)
Mod + H / L
Focus column left / right
Mod + J / K
Focus window up / down in column
Mod + F
Maximize column width
Mod + Q
Close focused window
Mod + Shift + H/L
Move column left / right
Mod + Shift + E
Quit niri session
Mod + B
Open Firefox browser
Mod + Mouse Scroll
Scroll through workspace columns
